# Identify-Groundwater-Infiltration-Types-in-caverns-using-LiDAR-and-MATLAB
Use MATLAB script to identify three flow infiltration pathways of groundwater using the stalactite morphology (aspect ratio and cross sectional area). Method developed by Dr. Kashif Mahmud and tested by Kathryn Brown and Matthew Kaspar using LiDAR data from the Natural Bridge Caverns.
